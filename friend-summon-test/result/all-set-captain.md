### 당신은 분위기를 띄우기 전에, 판부터 안 흔들리게 잡아주는 사람입니다.

### 왜 이렇게 보이냐면
단톡이 어수선해질 때도 당신이 한 번 정리하면 다들 바로 움직입니다. 누가 언제 가능한지, 어디가 제일 덜 꼬이는지, 뭘 먼저 해야 하는지 빠르게 잡아주는 힘이 있어서 친구들이 은근히 당신부터 찾게 돼요.

### 사람들이 느끼는 매력
- 말 길게 안 해도 다들 납득되는 정리력
- 헷갈리는 상황을 바로 실행 가능하게 바꾸는 감각
- 같이 있으면 일이 덜 꼬일 것 같은 안정감
